### 说明
本系统是由SpringMVC+hibernate搭建的基础框架。
主要目的是解决在项目开发中（特别是小型项目开发中）搭建框架的麻烦。
并使用本系统锻炼个人技术，如有不足。请指教。

### 技术选择
- JDK1.7.0_67
- 基于servlet3.0.1 使用tomcat7
- SpringMVC
- Hibernate，数据库连接池使用Druid
- SiteMesh （前端装饰）
- Shiro （apache的权限框架）
- 前端使用ACE前端框架（基于Bootstarp）
